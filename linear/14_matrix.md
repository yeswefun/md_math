    回忆: 标准单位向量

        二维空间中，有两个特殊的单位向量
            E1 = (1, 0), E2 = (0, 1)

          1 0       x       1        0        x       0       x
        ( 0 1 ) * ( y ) = ( 0 )x + ( 1 )y = ( 0 ) + ( y ) = ( y )

        = x*E1 + y*E2

            这就是(x, y)这个点坐标的定义

            在(1, 0)轴上的分量为x, 在(0, 1)轴上的分量为y

                  1 0       x       1        0
                ( 0 1 ) * ( y ) = ( 0 )x + ( 1 )y = x*E1 + y*E2
