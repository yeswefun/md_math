        A + B
        k * A

    矩阵加法: 对应分量相加
        A + B
    
    矩阵数乘: 所有分量同乘k

重点:
    矩阵是一组向量

    矩阵的基本运算性质
        A + B = B + A

        (A + B) + C = A + (B + C)

        k * (A + B) = k * A + k * B

        (c + k) * A = c * A + k * A

        (c * k) * A = c * (k * A)

        1 * A = A

        A + (-A) = O

        -1 * A = -A


    存在矩阵 O，满足 A + O = A
    存在矩阵 -A，满足 A + (-A) = O
        -A 唯一
        -A = -1 * A


    证明: k * (A + B) = k * A + k * B
        A ，B 必须同型
